⚠ ยังไม่ได้ตั้งค่า Firebase
ต้องใส่ค่า config ของโปรเจกต์ Firebase ของคุณเองในไฟล์นี้ก่อนใช้งานได้:
1. สร้างโปรเจกต์ที่ console.firebase.google.com
2. เปิดใช้ Firestore Database และ Storage
3. เพิ่ม Web App เพื่อรับค่า config
4. เปิดไฟล์นี้แล้วแก้ตัวแปร firebaseConfig ใกล้ด้านล่างของไฟล์ ใส่ค่าที่ได้ลงไป
5. ตั้งกฎความปลอดภัยของ Firestore/Storage ให้เหมาะสม (ดูคำแนะนำท้ายข้อความ)
